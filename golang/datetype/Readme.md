# golang 数据类型

<br />

## Go 的基本类型有

#### 布尔型  bool  

#### 字符串  string

#### 整型  int  int8  int16  int32  int64

#### 无符号整型 uint uint8 uint16 uint32 uint64 uintptr

#### byte // uint8 的别名

#### rune // int32 的别名 // 表示一个 Unicode 码点

#### 浮点型  float32 float64

#### 复数  complex64 complex128

<br />

### 没有明确初始值的变量声明会被赋予它们的 零值。

> 零值是：  
数值类型为 0，  
布尔类型为 false，  
字符串为 ""（空字符串）。
