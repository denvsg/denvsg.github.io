## 创建项目
1. 首先创建一个存放 Rust 代码的目录。
2.打开终端并输入命令创建 projects 目录，并在 projects 目录中为 “Hello, world!” 项目创建一个目录。
```shell
$ mkdir ~/projects
$ cd ~/projects
$ mkdir hello_world
$ cd hello_world
```

编写hello.rs
```rust
fn main() {
    println!("Hello, world!");
}
```
