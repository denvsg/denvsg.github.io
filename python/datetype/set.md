# 集合

### 集合（set）是一个无序的不重复元素序列。

可以使用大括号 { } 或者 set() 函数创建集合，注意：创建一个空集合必须用 set() 而不是 { }，因为 { } 是用来创建一个空字典。 如：

```python
set_1 = {1, 2, ...}
set_2 = set(0, 1)
```

## 集合的基本操作

### 1、添加元素

使用add函数，语法格式如下：

```python
s = set()
s.add("x")
```

### 2、删除元素

使用 remove 函数，语法格式如下：

```python
s = {"a", "c", "x", "y"}
s.remove("x")
```

> 注意：将元素 x 从集合 s 中移除，如果被删除元素不存在，则会发生错误。

### 3. 计算元素个数

使用len 函数，语法格式如下：

```python
s = {"a", "c", "x", "y"}
l = len(s)
print(l)
```
